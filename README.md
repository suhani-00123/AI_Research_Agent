# AI Research Agent
Project Description

This project is a simple AI Research Agent built using Python, Google Gemini API, and Tavily API. It allows users to ask questions and get AI-generated answers. If the question needs recent information, the project searches the web and uses the search results to generate a better response. It can also save the generated answer as a text report.
## Project Features
| Feature       | Description                                             |
| ------------- | ------------------------------------------------------- |
| Ask Questions | Users can enter any research question.                  |
| AI Response   | Gemini generates a clear answer.                        |
| Web Search    | Tavily searches for the latest information when needed. |
| Save Report   | The final response is saved in a text file.             |

## Technologies Used
| Technology        | Purpose                     |
| ----------------- | --------------------------- |
| Python            | Main programming language   |
| Google Gemini API | Generate AI responses       |
| Tavily API        | Search the web              |
| Google Colab      | Develop and run the project |
